# Local LLM Infrastructure

University project focused on understanding the infrastructure behind modern language-model applications.

The project covers the full process of running and deploying an LLM-based system:

* local model inference with Ollama,
* hardware compatibility analysis with `llmfit`,
* integration with Continue.dev in Visual Studio Code,
* direct API communication using `curl` and Python,
* Docker and Docker Compose containerization,
* optional MCP tool integration,
* deployment to an AWS EC2 instance,
* comparison of local and cloud performance.

## Technologies

* Ollama
* Continue.dev
* Python
* Docker
* Docker Compose
* Model Context Protocol
* AWS EC2

## Project Structure

The project is divided into four main phases:

1. Local model setup and inference
2. AI coding assistant integration
3. Docker containerization
4. Cloud deployment on AWS

The main goal is to understand how hardware, APIs, clients, containers, networking, and cloud infrastructure work together in an AI-powered application.
